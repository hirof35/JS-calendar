📅 究極のJSカレンダー (Ultimate Vanilla JS Calendar)
ライブラリを一切使用せず、Vanilla JS（純粋なJavaScript）のみで構築された高機能なカレンダーアプリケーションです。予定管理、通知、テーマ切り替えなど、モダンなWebアプリに必要な要素を網羅しています。

✨ 特徴
祝日連携: Holidays JP API から日本の祝日データを非同期取得し、カレンダー上に反映。

予定管理 (CRUD):

カレンダーの日付をクリックして予定（時刻・内容・カテゴリ）を保存。

LocalStorage を使用しているため、ブラウザを閉じてもデータが保持されます。

予定の完了チェック、削除、キーワード検索機能を搭載。

ドラッグ＆ドロップ: 予定一覧からカレンダーの日付へ予定をドラッグすることで、直感的な日付変更が可能。

デスクトップ通知: Web Notification API を使用。予定の10分前にブラウザからプッシュ通知を送信します。

マルチテーマ: 標準、ダークモード、サクラ（パステルピンク）の3種類を切り替え可能。

レスポンシブ設計: CSS変数（Variables）とFlexbox/Gridを用いた、保守性の高いレイアウト。

🛠 技術スタック
Language: JavaScript (ES6+)

Persistence: LocalStorage API

Communication: Fetch API (Holidays JP API)

Browser APIs: Notification API, Drag and Drop API

Styling: Modern CSS (Grid Layout, Custom Properties)

📐 ロジック解説
カレンダー生成アルゴリズム
Date オブジェクトを使用して、指定された月の「1日の曜日」と「月末の日付」を取得。Gridレイアウトを用いて、カレンダーの空セルの埋め合わせと日付の描画を動的に行っています。

ドラッグ＆ドロップの実装
dragstart で日付データをセットし、カレンダーセルの drop イベントで LocalStorage 内のキーを書き換えることで、データの移動を実現しています。

通知システム
setInterval を用いて1分ごとにバックグラウンドでチェック。現在時刻と予定時刻を比較し、指定時間（10分前）になった瞬間に一度だけ通知を発火させるフラグ管理を行っています。

🚀 使い方
このリポジトリをクローンまたはファイルをダウンロード。

index.html をブラウザで開くだけで動作します（サーバー環境不要）。

通知機能を利用する場合は、ブラウザからの通知許可プロンプトで「許可」を選択してください。

📁 ファイル構成
index.html: 全ての構造、スタイル、ロジックを含むスタンドアロンファイル。
